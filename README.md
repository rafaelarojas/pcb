# PCB

Esta atividade tem por objetivo desenvolver uma placa de circuito impresso capaz de receber um Raspberry Pi Pico W encaixado utilizando um dispositivo conector, além de um hardware para acionar uma interface de potência (relé e transistor), um dispositivo sinalizador (LED ou Buzzer) e um dispositivo de entrada (analógico ou digital). Esta placa deverá ser construída utilizando o software de CAD para placas eletrônicas utilizado no módulo vigente.
